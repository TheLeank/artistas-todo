Shabby docs

Get all tasks (GET):

http://localhost:8001/tasks/


Get task (GET):

http://localhost:8001/task/{id}/


Create task (POST):

http://localhost:8001/task/

Params: text (limited to 200 characters), body


Update task (PUT):

http://localhost:8001/task/{id}/

Params: text (limited to 200 characters), body


Delete task (DEL):

http://localhost:8001/task/{id}/
